# 🚀 实时接入正在进行的比赛 - 快速开始

## 3 步开始使用

### 第 1 步：打开日志页面
```
http://localhost:8080/static/logs.html
```

### 第 2 步：看到这个横幅
```
┌─────────────────────────────────────────────────────────┐
│ 🔴 正在进行的比赛                                        │
│ match_20260129_113744 | 10 个机器人 | 已运行 0分 15秒    │
│                                    [📡 接入实时日志]     │
└─────────────────────────────────────────────────────────┘
```

### 第 3 步：点击"接入实时日志"
- 自动加载机器人列表
- 选择要查看的机器人
- 数据每 2 秒自动刷新

**就这么简单！** 🎉

---

## 💡 使用技巧

### 切换机器人
在实时模式下，直接在下拉框中选择其他机器人，数据会立即切换。

### 查看历史比赛
比赛结束后（60 秒无数据），页面会自动提示并刷新，比赛会出现在历史列表中。

### 同时查看实时和历史
- 实时监控：http://localhost:8080/static/index.html
- 实时日志：http://localhost:8080/static/logs.html（接入模式）
- 历史日志：http://localhost:8080/static/logs.html（选择历史比赛）
