# 守门员测试场景说明

## 测试目标
专门测试守门员模块的行为，包括：
- 位置选择和移动
- 球的拦截和扑救
- 与球门柱的位置关系
- 不同射门角度的应对

## 测试步骤

### 1. 基础位置测试
- 球放在中线，观察守门员是否正确站位
- 球移动到不同位置，检查守门员跟随

### 2. 射门拦截测试
- 球以不同速度向球门移动
- 测试不同角度的射门
- 验证守门员的反应时间和动作

### 3. 边界情况测试
- 球在禁区边缘
- 球在球门区内
- 球在球门柱附近

### 4. 动态测试
- 球的轨迹预测
- 多次连续射门
- 不同球速的应对

## 评估指标
- 位置准确性
- 反应时间
- 成功拦截率
- 动作流畅性

## 使用方法
1. 启动SimRobot: `./SimRobot Config/Scenes/BH.ros3 -s Config/Scenarios/GoalkeeperTest`
2. 在GameController中设置为Playing状态
3. 手动放置球进行测试
4. 观察和记录守门员行为
